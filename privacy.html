Privacy Policy

Last Updated: October 2025

Because CineMuse is a purely client-side application (it does not use any server, database, or backend technology), we do not collect, store, or transmit any personal data about you.

Information We Use (Locally)

The application only uses the browser's built-in Local Storage to save certain non-personal data necessary for core features to work persistently on your device:

  • Watchlist: A list of movie IDs you manually save.
  • Theme Preference: Whether you prefer Dark Mode or Light Mode.
  • Cached Movies: The last viewed list of trending movies for offline access.

This data is stored exclusively on your device and is never shared with us or any third party.

External API

We use The Movie Database (TMDB) API to fetch movie information. TMDB has its own privacy policy which applies to the data they collect during your API requests.

You can clear all stored application data at any time by clearing your browser's Local Storage for this site.
